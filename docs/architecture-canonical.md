## Purpose
Define architecture in one place and enforce implementation conformance for an internal API-first platform.

## Principles
- `All infrastructure integrations must be implemented as adapters behind stable ports.`
- `Application code imports contract ports, not concrete infrastructure adapters.`
- `pico_app_id is the canonical app identity key across control-plane and runtime surfaces.`
- `Postgres is the durable system of record; vector and realtime systems can be swapped.`
- `Deployment must remain push-to-deploy through CI workflows and IaC-managed environments.`
- `Terraform-managed infrastructure must use a shared remote state backend instead of machine-local state.`
- `The frontend design system is mandatory and built on shared shadcn/ui-based primitives and tokens.`
